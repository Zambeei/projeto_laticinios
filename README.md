
# 🚀 <projeto_laticinios>

> **Projeto prático desenvolvido para o curso Técnico em Desenvolvimento de Sistemas - SENAI** > **Semestre:** 1º Semestre  
> **Disciplina/UC:** Lógica de Programação e Gestão Ágil

---

## 📋 Descrição do Projeto

< O gerente de uma empresa de laticínios, precisa de uma forma organizada e prática de verificar se a temperatura está correta para aquele tipo de alimento, em tempo real, e com alertas automáticos para avisar caso haja uma alteração na temperatura dos refrigeradores, sendo o causador do problema, pois os laticínios não estão sendo armazenados de forma correta, os fazendo estragar antes da validade.>
